Weather Data API

URL query format

URL format: http://127.0.0.1:5002/api/v1/station/date/

Get data from a particular station and date

URL example: http://127.0.0.1:5002/api/v1/10/1993-07-26

Get data from one station for all dates

URL example: http://127.0.0.1:5002/api/v1/10

Get data for one year on one station

URL example: http://127.0.0.1:5002/api/v1/yearly/10/1967
